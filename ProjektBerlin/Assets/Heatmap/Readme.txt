# Quick Start

Heatmaps are created via script, using the command Heatmap.CreateHeatmap().  Currently there is no user interface.

Example scripts are Tracker.cs and MouseClickDemo.cs, both of which have code that creates and takes a screenshot of a heatmap.

Online link to documentation:
http://paraboxstudios.com/heatmap/docs/annotated.html

Or go to Windows/Heatmap Documentation
